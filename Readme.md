A social meal planning Android app built with Jetpack Compose and Firebase.

## Features

- **Community Recipes** - Browse and save recipes shared by users
- **Meal Calendar** - Plan weekly meals with visual calendar
- **Create & Share** - Upload recipes with images to the community
- **Smart Filtering** - Filter by dietary preferences 
- **Comments** - Engage with the cooking community
- **Real-time Updates** - Instant synchronization across devices

## Tech Stack

- **Kotlin** + **Jetpack Compose**
- **Firebase** (Auth + Firestore)
- **Imgur API** (Image hosting)
- **Material Design 3**

## Setup

1. Clone the repository
2. Build and run in Android Studio

## Screens

- **Home** - Community recipe feed with filters
- **Saved** - Personal recipe collection
- **Calendar** - Weekly meal planner
- **Add** - Create new recipes
- **Profile** - User settings
